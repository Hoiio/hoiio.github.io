Modal

A modal is a child window that is layered over its parent window. Typically, the purpose is to display content from a separate source that can have some interaction without leaving the parent window. Child windows can provide information, interaction, or more.

Initializing a modal

Toggle the modal's hidden content via JavaScript : call an identifier of modal with a single line of JavaScript. 


<script>
$('.demo.modal').modal('show')
</script>

Attach events to another element, like button. 


<script>
// Clicking the demo button will show the demo modal.
$('.demo.modal')
  .modal('attach events', '.demo.button')
;
</script>

Options

Methods

Here are some useful methods that can be used with modal().

Method Description Example
Options .modal(options) Activates your content as a modal. Accepts an optional options object. 
$('.demo.modal').modal({
        closable: false
    })
Toggle .modal('toggle') Manually toggles a modal. 
$('.demo.modal').modal('toggle')
Show .modal('show') Manually opens a modal. 
$('.demo.modal').modal('show')
Hide .modal('hide') Manually hides a modal. 
$('.demo.modal').modal('hide')

Besides using an option with its value individually, you can call them as arguments of a setting method like show, hide, toggle : .modal('setting', 'options', value). Example: .modal('setting', 'closable', false)

Options

There are certain options which can be passed via JavaScript to customize the look and feel of the Modal Window. Following table lists the options.

Option Name Default Value Description
autofocus true When true, the first form input inside the modal will receive focus when shown. Set this to false to prevent this behavior.
allowMultiple false If set to true will not close other visible modals when opening a new one
closable true Setting to false will not allow you to close the modal by clicking on the dimmer
duration 400 Duration of animation
transition scale Named transition to use when animating menu in and out, full list can be found in ui transitions docs.

Multiple Modals

A modal can open a second modal. If you use allowMultiple: true parameter the second modal will be opened on top of the first modal. Otherwise the modal will be closed before the second modal is opened.

Modal #1

This is the modal 1. Click Next to go to modal 2.

Next
Modal #2

That's everything!

All Done 

<script>
$('.coupled.modal')
  .modal('setting', ' allowMultiple', false)
;
// Click the coupled button to show the first modal.
$('.first.modal')
  .modal('attach events', '.coupled.button')
;
// Click the action button in first modal to show the second modal.
$('.second.modal')
  .modal('attach events', '.first.modal .button')
;
</script>

Events

Following table lists the events to work with modal. These events may be used to hook into the function.

Event Description Example
onShow Is called when a modal starts to show. 
onShow: function () {
        // do something…                
    }
onVisible Is called after a modal has finished showing animating. 
onVisible: function () {
        // do something…                
    }
onHide Is called after a modal starts to hide. 
onHide: function () {
        // do something…                
    }
onHidden Is called after a modal has finished hiding animation. 
onHidden: function () {
        // do something…                
    }
onApprove Is called after an approve or ok button is pressed. If the function returns false, the modal will not hide. 
onApprove: function () {
        // do something…                
    }
onDeny Is called after a deny or cancel button is pressed. If the function returns false the modal will not hide. 
onDeny: function () {
        // do something…                
    }

Examples

If onDeny or onApprove returns false it will prevent the modal from closing.

$('.closable.false.modal')
  .modal({
    onVisible    : function(){
