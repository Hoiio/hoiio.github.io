Color

The Hoiio palette includes 5 main colors that applied to mostly web applications.

Neutral Colors

Includes colors from gray to full dark. In general, these colors are used for default content and help not to distract users, e.g. border, defaul text color or hovering action, etc.

Mid White#e8e8e8
Light Gray#eeeeee
Mid Gray#cccccc
Gray#767676
Light Black#555555
Black#1b1c1d

Primary color

Primary is a bright, warm blue that often used along with neutral colors.

Primary#008bd9
in Hover#009bf2
in Active#007bbf

Secondary color

Secondary color is extracted from neutral colors.

Secondary#e8e8e8
in Hover#eeeeee
in Active#cccccc

Action Colors

Action colors, which have inherent meaning, is defined to make users take certain action. For example, we use red color to communicate an error.
