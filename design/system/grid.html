Grid Framework

Grid Framework is a structure that allows for content to be stacked both vertically and horizontally in a consistent and easily manageable fashion. Additionally, grid system code is project-agnostic giving it a high degree of portability so that it may be adopted on new projects.

Because the design standards is firstly applied to Hoiio Apps, the screens we mainly support is for desktops and large desktops which min-width is 960px.

Columns

The grid system we use consists 16 columns. Columns have no gutters. The border and padding is simply included in the width of the grid columns.


<div class="ui grid">
  <div class="four wide column"></div>
  <div class="four wide column"></div>
  <div class="four wide column"></div>
  <div class="four wide column"></div>
  <div class="two wide column"></div>
  <div class="eight wide column"></div>
  <div class="six wide column"></div>
</div>

Rows

Rows are groups of columns which are aligned horizontally. The columns are nested inside of a row.


<div class="ui four column grid">
  <div class="row">
    <div class="column"></div>
    <div class="column"></div>
    <div class="column"></div>
  </div>
  <div class="column"></div>
  <div class="column"></div>
  <div class="column"></div>
  <div class="column"></div>
</div>

Forms

EXPAND GRID

The padding left and right of columns increased by 0.5rem.


<!-- Normal Grid -->                                
<div class="ui grid">
  <div class="three column row">
    <div class="column"></div>
    <div class="column"></div>
    <div class="column"></div>
  </div>
</div>  

<!-- Expand Grid -->                                 
<div class="ui expand grid">
  <div class="three column row">
    <div class="column"></div>
    <div class="column"></div>
    <div class="column"></div>
  </div>
</div>

CONTENT CENTERING

By adding a centered class, the grid will center the column contents inside.


<div class="ui two column centered grid">
  <div class="column"></div>
  <div class="four column centered row">
    <div class="column"></div>
    <div class="column"></div>
  </div>
</div>

CONTENT Clearing

Row wrappers will automatically clear previous columns, making them useful when using floated variations.


<div class="ui grid">
    <!-- Floated -->         
    <div class="four column row">
        <div class="left floated column"></div>
        <div class="right floated column"></div>
    </div>
    
    <!-- Non Floated -->     
    <div class="row">
        <div class="three wide column"></div>
        <div class="eight wide column"></div>
        <div class="five wide column"></div>
    </div>
</div>
