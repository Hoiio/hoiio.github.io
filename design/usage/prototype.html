Unlike HTML which begins with no assumptions on what a tag means, Semantic definitions begins with some baseline assumptions. A "standard" element describes in css a neutral form of an element with the most common qualities.

Start with UI

UI is a special class name added to html elements that correspond with ui definitions. This is used for containing css specificity, and distinguishing parts of an element from elements themselves.

Company Department
Staff

Usage : use divider with icon.


<div class="ui breadcrumb">
<a class="section">Company</a>
<i class="divider icon angle right"></i>
<a class="section">Department</a>
<i class="divider icon angle right"></i>
<div class="section">Staff</div>
</div>
