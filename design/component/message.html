Message

Definition Types Variations

A message displays information that explains nearby content.

Well done!

You successfully read this important alert message.

Usage :
- Add classes ui, checkbox to the <div> element.
- Include an <input> with checkbox type and a <label> element. 


<div class="ui message">
  <div class="header">
    Well done!
  </div>
  <p>You successfully read this important alert message.</p>
</div>

A list inside

A message contains a list inside. Reference to list at component / list

New Site Features
You can now have cover images on blog pages
Drafts will now auto-save while writing

Usage : put ui list (bulleted or ordered) in ui message.


<div class="ui message">
  <div class="header">New Site Features</div>
  <div class="ui bulleted list">
    <div class="item">You can now have cover images on blog pages</div>
    <div class="item">Drafts will now auto-save while writing</div>
  </div>
</div>

Icon Message

A message contains an icon inside. In case, a message looks like an item but is used to announce information.

Have you heard about our mailing list?

Get the best news in your e-mail every day.

Usage : prefix icon to message and put an item in ui message.


<div class="ui icon message">
  <i class="inbox icon"></i>
  <div class="content">
    <div class="header">Have you heard about our mailing list?</div>
    <p>Get the best news in your e-mail every day.</p>
  </div>
</div>

Dismissible Message

Build on any message by adding a javascript and remove button.

Welcome back!

This is a special notification which you can dismiss if you're bored with it.

Usage : use remove icon and add the behavior defined by Javascript,


<div class="ui message">
  <i class="remove icon"></i>
  <div class="header">
    Welcome back!
  </div>
  <p>This is a special notification which you can dismiss if you're bored with it.</p>
</div>

<!-- Javascript -->
<script>
$('.message .remove')
  .on('click', function() {
    $(this)
      .closest('.message')
      .transition('fade')
    ;
  })
;
</script>

Attached

A message can be formatted to attach itself to other content.

Welcome to our site!

Fill out the form below to sign-up for a new account

Submit
Already signed up? Login here instead.

Usage : insert a class .attached before the class .message.


<div class="ui attached message"></div>
<form class="ui form attached segment">
    <div class="two fields">
        <div class="field"></div>
        <div class="field"></div>
    </div>
    <div class="field"></div>
    <div class="field"></div>
    <div class="field"></div>
    <div class="ui primary submit button">Submit</div>
</form>
<div class="ui bottom attached warning message"></div>

Floated

A message can float above content that it is related to.

I'm flying!

Usage : insert a class .floating before the class .message.


<div class="ui floating message">
  <p>I'm flying!</p>
</div>

Compact

A message takes width according to its content automatically.

Get all the best inventions in your e-mail every day. Sign up now!

Usage : insert a class .compact before the class .message.


<div class="ui compact message">
    <p>Get all the best inventions in your e-mail every day. Sign up now!</p>
</div>
