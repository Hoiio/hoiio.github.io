Label

A label provides additional information about something that is often used associate to a form.

This is a label
23
Tag 

<div class="ui label">
  This is a label
</div>

States

Disabled

Disabled label

Usage : prefix disabled to label.


<div class="ui disabled label">
  Disabled label
</div>

Variation

Colors

Primary Primary Info Success Error Warning

Usage : prefix classes of colors to label.

<a class="ui info label">Info</a>
<a class="ui success label">Success</a>
<a class="ui error label">Error</a>
<a class="ui warning label">Warning</a>

Types

Tag

A label can display as a tag which attached to someone or something for the purpose of identification or to give other information.

New Upcoming Featured

Usage : prefix tag to label.


<a class="ui tag label">New</a>
<a class="ui error tag label">Upcoming</a>
<a class="ui info tag label">Featured</a>

Corner

A label can position itself in the corner of an element.

Usage : prefix left corner or right corner to label.


<div class="ui fluid image">
      <a href="#" class="ui left corner label"><i class="heart icon"></i></a>
      <img src="image.jpeg">
</div>

<div class="ui fluid image">
      <a href="#" class="ui error right corner label"><i class="floopy icon"></i></a>
      <img src="image.jpeg">
</div>

Tooltip

A label can be display as a tooltip

Please enter a value
Please enter a value
That name is taken!
Your password must be 6 characters or more

Usage : append tooltip, tooltip below, left tooltip, right tooltip to label for up, down, left and right point alternately. Note that to display left or right tooltip, append inline to the field. For showing background colors, using types of colors.


<div class="field">
    <input type="text" placeholder="First name">
    <div class="ui label tooltip">Please enter a value</div>
</div>

<div class="field">
    <div class="ui label tooltip below">Please enter a value</div>
    <input type="text" placeholder="Last Name">
</div>

<div class="field inline">
    <input type="text" placeholder="Username">
    <div class="ui label left tooltip">That name is taken!</div>
</div>

<div class="field inline">
    <div class="ui label right tooltip">Your password must be 6 characters or more</div>
    <input type="password" placeholder="Password">
</div>
