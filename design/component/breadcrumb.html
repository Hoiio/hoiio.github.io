Breadcrumb

Definition States Variations

Breadcrumbs are a great way to show hierarchy-based information for a website. They can indicate the current page's location within a navigational hierarchy. Basically, a breadcrumb is an inline unordered list with a class of .breadcrumb.

Company
/
Department
/
Staff

Usage : add classes .ui, .breadcrumb to the <div> element. 


<div class="ui breadcrumb">
    ...
</div>

A breadcrumb contains divider, section and link.

Section

Company
/
Staff

Usage : add a <div> element with class .section into .breadcrumb.


<div class="ui breadcrumb">
  <div class="section">Company</div>
  <div class="divider"> / </div>
  <div class="section">Staff</div>
</div>
