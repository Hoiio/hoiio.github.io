Checkbox

Definition States Types

Check boxes are used when we want to let user select one or more options from a set of alternatives. If only one option is to be selected, we use radio buttons instead. A checkbox does not require Javascript to function; it works with an input with a label.

Usage :
- Add classes .ui, .checkbox to the <div> element.
- Include an <input> with checkbox type and a <label> element. 


<div class="ui checkbox">
  <input type="checkbox" name="example">
  <label>Make my profile visible</label>
</div>

Disabled

A checkbox is unable to change states.

Usage : insert a class .disabled before the class .checkbox.


<div class="ui disabled checkbox">
  <input type="checkbox" name="example">
  <label>Disabled</label>
</div>

Checked

Usage :
- Insert a class .checked before the class .checkbox.
- Add checked = "" to the input element, to help with accessibility. 


<div class="ui checked checkbox">
  <input type="checkbox" checked="">
  <label>Active</label>
</div>

Click on label

Clicking on a label next to box can check this box. It only works with checkbox and radio.

Usage :
- Add a .hidden class and id attribute with its name to <input> element.
- Add into label a for attribute which name is same as id attribute's. 


<div class="ui checkbox">
  <input type="checkbox" id="group" name="example" class="hidden">
  <label for="group">Remember me</label>
</div>

Radio

A checkbox is formed as a radio element. This means it is an exclusive option.

Usage :
- Insert a class .radio to the class .checkbox.
- Replace the checkbox type with a radio type. 


<div class="ui radio checkbox">
  <input type="radio" name="radio">
  <label>Radio choice</label>
</div>

Slider

A checkbox is formatted to emphasize the current selection state.

Usage : Insert a class .slider to the class .checkbox. 


<div class="ui slider checkbox">
  <input type="checkbox" name="newsletter">
  <label>Accept terms and conditions</label>
</div>

Toggle

A checkbox is formatted to emphasize the current selection state.

Usage : Insert a class .toggle to the class .checkbox. 


<div class="ui slider checkbox">
  <input type="checkbox" name="newsletter">
  <label>Accept terms and conditions</label>
</div>
