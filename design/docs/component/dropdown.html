Dropdown

Definition Content States Types Variations

A dropdown is a toggleable menu that allows the user to choose one value from a predefined list.

Dropdown
Choice 1 Choice 2 Choice 3

Usage :
- Add classes .ui, .dropdown to the <div> element.
- Wrap menu in a .dropdown. Use <i class="dropdown icon"></i> to act as an indicator. 


<div class="ui dropdown">
  Dropdown <i class="dropdown icon"></i>
  <div class="menu">
    <a href="#" class="item">...</a>
  </div>
</div>
Dropdown
Choice 1 Choice 2 Choice 3

Usage : add a class .button to the class .dropdown. 


<div class="ui dropdown button">
  Dropdown <i class="dropdown icon"></i>
  <div class="menu">
    <a href="#" class="item">...</a>
  </div>
</div>

A dropdown menu can contain a header, a description, a divider, an icon.

Header

A dropdown menu can contain a header.

Select an app

Marketing

Usage : insert a <div> element with a class .header into the menu.


<div class="ui dropdown button">
  Select an app <i class="dropdown icon"></i>
  <div class="menu">
    <!-- Header -->
    <div class="header">
        <i class="block layout icon"></i>
        Human Resource
    </div>
    <a href="#" class="item"></a>
  </div>    
</div>

  </div>
</div>
